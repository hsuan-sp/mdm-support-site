## 術語定義

**Global HTTP Proxy (全域 HTTP 代理)** 是一種強制性的網路設定，要求裝置的所有 HTTP/HTTPS 流量都必須經過指定的代理伺服器。

這在組織管理上常用於：

* **內容過濾**：在網關層級 (Gateway Level) 直接阻擋色情、賭博等不當網站。
* **行為稽核**：集中紀錄所有使用者的上網足跡與瀏覽歷史。
* **防止繞過**：無論裝置在校內或校外，只要連上網，過濾規則即刻生效。

**技術限制**：雖然安全性高，但可能會造成部分高流量 App (如視訊會議) 延遲，或導致不支援 Proxy 的 App 無法連線。

## 白話文比喻

這是網路流量的 **「強制安檢站」**。

就像進出機場的車輛都被引導走同一條車道，所有要進出的資料封包（車輛）都必須先開進這個檢查站受檢。

雖然這樣可以有效攔截「違禁品」（如色情網站），但也因為每台車都要停下來檢查，很容易在尖峰時刻造成 **「大塞車」**（網速變慢）。
