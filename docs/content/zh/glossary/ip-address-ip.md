## 術語定義

**IP Address (網際網路通訊協定相關位址)** 是裝置在網路上的唯一識別碼與定位地址。

主要分為以下類型：

* **固定 IP (Static IP)** ：手動設定，永久不變，如同買斷的門牌。
* **動態 IP (Dynamic IP / DHCP)** ：由伺服器自動分配，會隨時間或重新連線而改變。
* **虛擬 IP (Private IP)** ：僅在區域網路內有效（如 `192.168.x.x`），外部無法直接存取。

MDM 伺服器依賴準確的 IP 位址來與裝置通訊及記錄連線來源。

## 白話文比喻

這是網路世界的 **「門牌號碼」** 。

郵差（數據資料）必須知道準確的地址（IP），才能把信（網頁內容、指令）準確地送到你家（iPad）。

有些地址是 **「自購豪宅」** （固定 IP），永遠都是你的，朋友隨時找得到；有些像是 **「旅館房間」** （動態 IP），你入住時給你一個號碼，退房後這個號碼就會換給別人用。
