## 術語定義

**"Install Later" (稍後安裝)** 是 Apple 系統更新的一種智慧部署策略，旨在將對使用者的干擾降到最低。

其運作細節如下：

* **背景下載** ：裝置會在背景靜默下載龐大的更新檔，使用者可繼續上課或工作，完全不受影響。
* **排程安裝** ：實際需要重新開機的安裝動作，會被安排在裝置閒置時段（通常是深夜）自動執行。
* **彈性選擇** ：如果使用者希望立即更新，也可以在檔案下載完成後手動觸發。

這確保了安全性更新能及時部署，又不會在課堂中途突然強制重開機，打斷學生的學習。

## 白話文比喻

這就像是 **「預約夜間施工」** 。

IT 部門先把 **「建材」** （更新檔）偷偷搬到工地（iPad）旁放好，這時候學生還在裡面上課，完全不會被吵到。

等到學生放學回家、睡覺了， **「施工隊」** （安裝程序）才進場敲敲打打（重開機更新）。第二天早上學生上學時，教室（系統）已經翻修一新，隨時可以用。
