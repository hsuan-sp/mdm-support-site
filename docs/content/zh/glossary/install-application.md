## 術語定義

**"Install Application"** 是 MDM 伺服器發送給裝置的標準指令，用以啟動軟體部署流程。

其運作流程如下：

1. **發送指令** ：MDM 發出包含 App ID (Bundle ID) 的指令。

2. **裝置接收** ：裝置確認收到指令並檢查網路狀態。

3. **執行安裝** ：裝置聯繫 **App Store** (或 VPP 伺服器) 下載並安裝指定的軟體。

4. **回報狀態** ：安裝完成後，回報成功訊息給 MDM。

## 白話文比喻

這就像是一張 **「數位採購清單」** 。

老師不需要走到每個學生的座位旁，一台一台幫忙點擊下載。

老師只需要在系統上開出一張清單（發送指令），所有 iPad 收到後，就會自己跑去 **「數位圖書館」** （App Store）把上課要用的工具通通領回來裝好。
