## 術語定義***Rapid Security Response (RSR / 快速安全回應）**是 Apple 設計的一種敏捷更新機制，專門用來修補緊急的安全漏洞。

它與一般更新的不同之處：

*  **小巧快速**：RSR 檔案很小，下載與安裝速度比完整的系統更新快非常多。

*  **插隊修補**：它會在兩個主要版本（例如 iOS 16.4 和 16.5）之間釋出，專門堵住突然發現的資安缺口。

*  **低干擾**：安裝後通常只需要快速重啟，甚至針對 Safari 的更新不需要重開機，大幅減少對上課的影響。

*  **識別方式**：版本號後面會帶一個英文字母，例如 `iOS 16.4.1 (a)`。

## 白話文比喻

這是系統的**「緊急 OK 繃」**。

*  **大手術 （系統更新）**：以前不管傷口多小，都要進手術房全身麻醉（完整更新），還得停工好幾個小時。

*  **OK 繃 (RSR)**：現在發現一個小傷口（漏洞），Apple 護士直接過來貼個**OK 繃**就好了。過程只要幾分鐘，貼完馬上就能繼續跑跳，不用躺在病床上等半天。
