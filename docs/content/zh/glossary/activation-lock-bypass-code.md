## 術語定義

**Activation Lock Bypass Code (啟用鎖定繞過代碼)** 是一組由 MDM 伺服器自動從 **受監管 (Supervised)** 裝置中收集的唯一英數隨機字串。

這組代碼對於裝置的生命週期管理至關重要：

- 當使用者離開組織，卻忘記登出其個人的 **Apple 帳號** （導致裝置處於「啟動鎖定」狀態）時。
- IT 管理員可以在裝置的密碼欄位中輸入這組特定代碼，以「繞過」原本的帳號鎖定。
- 如此一來，裝置便能順利被抹除，並重新分配給下一位使用者。

## 白話文比喻

這是機構專屬硬體的 **「緊急萬能鑰匙」** 。

如果有人離開學校時忘了把 iPad 解鎖，資訊組長不需要去猜測學生的個人密碼。管理員只需從 MDM 系統中提取這串代碼並輸入裝置，原本的「防盜鎖」就會立即解除。這台 iPad 就能被徹底清空並交給新學生使用。
