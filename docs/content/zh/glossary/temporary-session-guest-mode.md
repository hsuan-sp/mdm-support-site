## 術語定義

**Temporary Session (臨時工作階段)** ，常稱為 **Guest Mode (訪客模式)** ，是 **Shared iPad** 環境中的一種功能。

使用流程：

- **登入** ：使用者在登入畫面點擊「訪客」，無需輸入使用者名稱或密碼。

- **使用** ：在工作階段期間，使用者可以完整使用 App 與 Safari 瀏覽器。
- **登出** ：當使用者登出時，該工作階段的加密金鑰會被銷毀。
- **清除** ：所有資料——瀏覽紀錄、文件、照片、快取——都會被「瞬間且永久」地清除。

## 白話文比喻

請把它想像成 **「圖書館的公用電腦」** 。

- **免卡使用** ：你不需要辦借書證，坐下來就可以開始用。

- **自動清理** ：但是，當你站起來 **「登出」** 的那一刻，鍵盤會自動消毒，電腦會自動 **碎掉** 你留在桌上的所有紙張。
- **隱私** ：這確保了下一個人絕對看不到你剛剛做了什麼，而且 iPad 對每個人來說都是乾淨、快速的。
