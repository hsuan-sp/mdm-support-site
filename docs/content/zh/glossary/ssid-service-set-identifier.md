## 術語定義

**SSID (Service Set Identifier / 服務集識別碼)** 是無線網路路由器廣播出來的網路名稱（技術術語）。

學校常見設定：

* **School-Secure**：加密網路，使用 WPA 2/WPA 3-Enterprise (802.1X)，專供受管裝置使用。
* **School-Guest**：訪客網路（開放或密碼保護），與學校內部伺服器隔離。
* **隱藏 SSID**：不廣播名稱的網路（安全性較低，現代已不推薦使用）。

## 白話文比喻

SSID 就好比是 **「教室門上的班級牌」**。

* **辨識**：當你走在走廊上，你會看到「五年一班」、「五年二班」、「教職員室」的掛牌。
* **連線**：你會選擇與你身份相符的房間進去。就像你的 iPad 會選擇連上正確的 Wi-Fi**「房間」**(SSID)，以取得正確的網路權限。
