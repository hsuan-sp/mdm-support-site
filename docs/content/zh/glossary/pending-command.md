## 術語定義

**Pending Command (待處理指令)** 指的是 MDM 已經送出，但裝置尚未執行或回報結果的指令。

它的特性包括：

* **暫存狀態**：指令已進入 APNs 雲端佇列，正在等待裝置上線領取。
* **常見原因**：裝置關機、休眠、斷網（Wi-Fi 不穩），或是裝置忙碌中。
* **自動重試**：一旦裝置重新連上網路並通報 APNs，它就會立刻收到通知並下載這些積壓的指令來執行。

## 白話文比喻

這是郵局的 **「存局候領包裹」**。

* **寄出**：MDM 已經把包裹（指令）寄出去了。
* **未達**：因為收件人（裝置）不在家或是去旅行（關機/斷網），包裹暫時放在郵局。
* **領取**：等收件人一回到家（連網），郵差就會立刻把累積的包裹全部送過去。
