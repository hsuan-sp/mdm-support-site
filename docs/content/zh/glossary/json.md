## 術語定義

** JSON (JavaScript Object Notation) ** 是一種輕量級的資料交換格式，其特色在於易於人類閱讀與撰寫，同時也便於機器進行解析與產生。在 MDM 管理領域，JSON 經常被用來撰寫進階的配置架構 (Configuration Schemas)，或是在不同系統之間透過 API 傳輸資料。其結構是建立在「鍵值對 (Key-Value pairs)」的基礎之上。

## 白話文比喻

可以將其想像成一種 ** 「數位標準化表單」 ** 。

它是電腦系統之間溝通時，一種非常有條理的資訊傳遞方式。舉例來說，與其傳送一段雜亂無章的文字段落，系統會傳送結構工整的內容，例如：`{ "姓名": "學生 A", "年級": 5 }`。

因為格式始終保持一致，接收資訊的電腦能準確知道在哪個位置找到每一項資料，絕對不會發生看錯行或理解錯誤的情況。
