## 術語定義

**Activation Lock (啟用鎖定)** 是 Apple 的防盜機制，旨在防止遺失或遭竊的裝置被未經授權使用。它是 **「尋找 (Find My)」** 服務的一部分。

運作方式：

- 當開啟 **「尋找」** 功能時，會自動啟用此鎖定。
- 裝置重置後，必須輸入原使用者的 Apple ID 與密碼才能重新啟用。

雖然這對防盜非常有效，但若學生或員工離職時未登出，對組織來說是一大挑戰。在這種情況下，管理員必須使用 **Activation Lock Bypass Code (啟用鎖定略過代碼)** 或請求 Apple 支援協助解鎖。

## 白話文比喻

把這想像成裝置的 **「防盜死結」** 。

即使小偷把 iPad 整個清空重灌，開機畫面還是會顯示一道鎖，要求輸入你專屬的密碼。這讓裝置除了原主之外，對其他人都毫無用處。

但是，如果學生畢業後忘記解鎖，它就會變成 **「殭屍裝置」** ——學校留著硬體卻無法給下一個人使用，除非 IT 人員有 **「萬能鑰匙 (Override Key)」** 。
