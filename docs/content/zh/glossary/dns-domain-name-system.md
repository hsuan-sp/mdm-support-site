## 術語定義

**DNS (Domain Name System)** 是網際網路的電話簿服務。

其功能是將人類容易記憶的 **網址** （如 `apple.com`）翻譯成電腦看得懂的 **IP 位址** （如 `17.253.144.10`）。

MDM 可以對受管裝置強制指定使用特定的 **DNS 伺服器** ，這通常用於：

* **內容過濾** ：阻擋已知的惡意網站或不當內容。
* **存取加速** ：改善校內或企業內網的連線速度。
* **資安防護** ：防止裝置被惡意 DNS 劫持。

## 白話文比喻

這是網路世界的 **「自動查號台」** 。

你知道常去的店家叫做「麥當勞」（網址），但電話機（電腦）只認得「02-12345678」（IP 位址）。

所以每次你要打電話（上網），都要先問查號台（DNS）：「請問麥當勞幾號？」。查號台告訴你號碼後，你的電腦才能真正撥通電話連上網站。
