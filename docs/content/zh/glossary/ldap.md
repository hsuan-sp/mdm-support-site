## 術語定義

**Lightweight Directory Access Protocol (LDAP)** 是一種開放標準的網路協定，用於存取和維護分散式目錄資訊服務。

在 MDM 管理中的常見應用包括：

* **使用者查找**：讓 MDM 能連接到學校的 **Active Directory (AD)** 伺服器搜尋教職員資料。

* **群組同步**：自動將「五年級導師」或「行政人員」等群組名單同步到 MDM，以便精準派發 App。
* **身分驗證**：在使用者註冊裝置時，向後端資料庫確認帳號密碼是否正確。

## 白話文比喻

這是不同電腦系統之間的 **「共通語言」**。

就像台灣人和美國人講英文才能溝通一樣，你的 MDM 伺服器和學校的人事資料庫（Email 伺服器）原本互不相通。

透過 **LDAP** 這個語言，MDM 可以開口問資料庫：「請問『王小明』是幾年級的學生？」或是「這組密碼對嗎？」，資料庫聽得懂並回答，雙方才能合作。
