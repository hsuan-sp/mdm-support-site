## 術語定義

**Student Information System (SIS / 學生資訊系統)** 是學校用來管理學生資料的權威資料庫。

主要功能：

* **記錄**：儲存出席紀錄、成績、健康狀況與家長聯絡資訊。
* **整合**：與 **Apple School Manager (ASM)** 同步，自動提供最準確的「管理式 Apple 帳號」名單與班級名冊。

* **自動化**：當新轉學生被登錄到 SIS 後，系統會自動在學校的科技系統中建立他的數位身分，並將他分派到正確的班級群組。

## 白話文比喻

這是 **「教務處的數位大腦」**。

* **核心**：它是學校的中樞神經，知道每個學生的名字、在哪一班、爸媽是誰。
* **連動**：當我們把這個 **「大腦」** 連接到學校的 iPad 系統時，一切都能保持同步。如果有新生報到，大腦會通知 iPad 系統自動發給他帳號，完全不需要老師手動填寫任何表格。
