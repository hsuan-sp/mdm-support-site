## 術語定義

**Recovery Mode (復原模式)** 是 Apple 裝置內建的一個獨立開機磁區，專門用於緊急維護與故障排除，又稱為 **「macOS 復原」** 。

這裡提供的工具包括：

* **磁碟工具 (Disk Utility)** ：修復硬碟錯誤或格式化磁碟。
* **重新安裝 macOS** ：下載並安裝全新的作業系統。
* **時光機還原** ：從 Time Machine 備份中恢復資料。
* **安全性公用程式** ：修改開機安全策略（例如允許從隨身碟開機）。

## 白話文比喻

這是電腦的 **「緊急手術室」** 。

* **一般病房 (正常開機)** ：這是電腦平常工作的地方。
* **送急診 (復原模式)** ：當電腦病重到連開機都開不了（系統損毀），我們就把它推推這間 **「手術室」** 。
* **醫生** ：這裡面住著幾位專科醫生（內建工具），有的負責換大腦（重灌系統），有的負責修補器官（修復磁碟）。這是救活死機電腦的最後手段。
