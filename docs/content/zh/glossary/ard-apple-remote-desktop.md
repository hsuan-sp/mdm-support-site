## 術語定義***Apple Remote Desktop (ARD)**是一套專為 macOS 設計的強大桌面管理應用程式，適用於區域網路管理。

雖然 MDM 負責廣泛且持久的設定，但 ARD 專精於**即時性任務 (Real-time Tasks)**：

*  **螢幕監控**：在教室或電腦教室中同時查看多個 Mac 螢幕。

*  **遠端協助**：接管使用者的鍵盤與滑鼠以提供技術支援。

*  **軟體分發**：將大型安裝包或檔案快速推送到一群學生筆電中。

*  **執行指令**：在選定的目標 Mac 上執行遠端 Shell 腳本。

## 白話文比喻

把這想像成 IT 人員的**「即時控制台」**。

如果 MDM 像是貼在佈告欄上的校規（派送設定），ARD 就像是老師站在教室裡親自指導。IT 人員可以看到你的螢幕，移動你的滑鼠來示範操作，或是**「廣播」**一份檔案到教室裡的所有電腦上，瞬間完成。
