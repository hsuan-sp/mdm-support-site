**關閉個人熱點**：若任何一方開啟了「個人熱點」，AirDrop 通常會無法運作。 **AirPlay 投影問題排查**： AirPlay 依賴 **區網連線**。 1. **Wi-Fi 網路一致性**：最常見的原因是 iPad 與 Apple TV (或大屏) **連到了不同的 Wi-Fi 網路**。請確保兩者在同一個無線網段下。 2. **跨網段/VLAN**：若學校網路有切割 年或未來），會導致 SSL 憑證被判定為過期或尚未生效。 * **解法**：前往 **「設定」&gt;「一般」&gt;「日期與時間」**，確認時間正確，或開啟「自動設定」。 2. **憑證信任設定**： * 若學校使用 SSL 內容過濾或特定 Wi-Fi 認證，需手動信任根憑證。 * **解法**：前往 **「設定」&gt;「一般」&gt;「關於本機」&gt;「憑證信任設定」**，將相關的根憑證勾选為 **「啟用」**。 **問題描述**：設定圖示上有個消不掉的紅點，點進去顯示「更新 Apple ID 設定」，要求輸入密碼，但輸入後沒多久又跑出來。 **原因**： 這通常發生在**更換過 Apple ID 密碼**後，或者**VPP 授權被誤指派給了使用者 (User-Based) 而非裝置 設定改為 **Device-Based (裝置指派)**，即可避免 iPad 一直向使用者已要 Apple ID 授權。 **解決方案**： 在 Jamf Pro 的 **限制描述檔 (Restrictions Payload)** 中可以設定。 1. 編輯「學生限制描述檔」。 2. 前往 **「一般 (General)」** 頁籤（或視 Jamf 版本位於不同子分類）。 3. 勾選 **「強制設定日期與時間 (Force Date and Time automatic adjustment)」** 或是限制 **「允許修改日期與時間 (Allow modifying date and time)」** (取消勾選)。 4. **注意**：此功能需在 iPad 有定位服務或連網時才能準確校時。啟用此限制後，學生將無法手動調整時間。 **人損判斷標準**： 只要裝置外觀出現非正常使用下產生的物理損傷，皆會被原廠判定為**人為損壞 (人損)**。常見情況包括： * 螢幕玻璃碎裂、液晶顯示異常（出現線條、漏液）。 * 機身彎曲、變形、凹陷、有明顯撞擊痕跡。 * 充電孔或耳機孔有液體侵入或腐蝕跡象。 * 按鍵凹陷或無法按壓。 **維修報價流程（以精進方案平板為例）**： 精進方案的載具擁有四年保固，但**不包含人損**。人損件的處理方式如下： 1.
