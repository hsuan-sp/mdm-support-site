# Q: 寒暑假長期不使用，平板該如何保存電池才不會損壞？

# Answer

**根據 Apple 官方技術規範，長期存放裝置時應「維持約 50% 電量」、「完全關機」並置於「陰涼環境」。**
**這能有效避免電池因深度放電而永久失效。**

鋰離子電池的化學特性決定了存放方式將直接影響其壽命。
對於學校在長假期間存放大量 iPad 的需求，正確的標準作業程序 (SOP) 可避免開學後出現大規模「死機」現象。

**長期存放的三大關鍵指標：**

| 存放狀態 | 影響結果 | 官方技術原因 |
| :--- | :--- | :--- |
| **0% (完全沒電)** | **深度放電** | 電池可能進入深度放電狀態，導致**永久損壞**，未來將無法再充電。 |
| **50% (理想電量)** | **狀態穩定** | 電池處於最穩定的化學平衡點，自放電風險最低。 |
| **100% (滿電)** | **容量減損** | 電池長期維持高壓狀態，將導致部分蓄電量永久喪失，縮短續航力。 |

---

**建議執行 SOP：**

**第一階段：放假前（封存作業）**
1.  **電量校準**：將平板充電或放電至 **50% 左右**（40% 至 60% 之間皆可）。
2.  **完全關機**：長按電源鍵並執行「滑動來關機」。
    切勿僅按電源鍵進入休眠（螢幕黑掉），因為背景程序與 MDM 簽到仍會持續耗電。
3.  **環境挑選**：存放於低於 32°C、乾燥且陰涼處。
    **嚴禁**放置於陽光直射的窗邊或無通風的悶熱鐵櫃中。
4.  **斷開電源**：確認充電車總電源已拔除。
    若持續插電，裝置會因維持在 100% 滿電狀態而加速電池老化。

**第二階段：假期中（期中抽檢）**
*   **針對精進方案第四年裝置 (iPad 9)**：由於電池已使用近四年，其自放電速度會比新機快。
    建議每 3 個月（或寒暑假中期）抽樣檢查電量，若低於 20% 應補電至 50% 後再次關機。

**第三階段：開學前（復課準備）**
1.  **預熱充電**：裝置長期存放後可能處於低電量狀態，
    開機前請先使用 **20W 原裝電源轉接器** 充電至少 20 分鐘。
2.  **資產簽到**：全數開機連網，讓裝置向 Jamf Pro 完成資產簽到 (Inventory Update)，
    確保配置描述檔與 App 處於最新狀態。

---

**實務建議：常見風險提示**
*   **電池膨脹 (Swelling)**：若存放環境溫度過高（如夏天關閉空調的教室），
    會大幅增加電池膨脹擠壓螢幕的風險。
*   **無法喚醒**：若開學後發現裝置插電無反應，請嘗試持續充電 12-24 小時。
    若仍無效，通常代表電池已因過度放電導致保護電路鎖死，需報修處理。

**官方資料依據**：
本建議參考 Apple 支援文件：[「充分發揮電池效能：存放裝置」](https://www.apple.com/tw/batteries/maximizing-performance/)。
