# Q: macOS 設備管理：如何透過 MDM 控管外接磁碟 (USB/Thunderbolt) 的讀寫權限以防止資料外洩？

# Answer

**自 macOS 15 起，Apple 引進了全新的「媒體管理 (Media Management)」與「外部儲存裝置」控管架構，**
**讓管理員能精細控制資料進出。**

這在電腦教室或行政電腦管理上至關重要，可防止學生透過 USB 隨身碟複製考卷、竊取個資或植入惡意軟體。

**Jamf Pro 實作策略：**
1.  **唯讀模式 (Read-Only)**：
    *   設定外接磁碟為「唯讀」。學生可以從隨身碟讀取教材，但**無法將電腦內的檔案複製出去**。
2.  **完全封鎖 (Block)**：
    *   針對高機密行政電腦，可設定「禁止掛載」任何外部儲存裝置。
3.  **Thunderbolt 配件安全性**：
    *   針對 Apple Silicon (M系列) 機型，可限制新連接的 Thunderbolt 裝置必須經過「使用者核准」
        或「管理員授權」才能進行數據傳輸，防止惡意周邊攻擊。
