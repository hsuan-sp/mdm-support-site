# Q: 【終極救援】iPad 卡在白蘋果、開機循環或更新失敗，如何透過「復原模式」與「DFU」刷機自救？

# Answer

**當 iPad 遇到嚴重軟體故障（如更新中斷導致死機、無限重開機）且 MDM 指令無效時，**
**需透過電腦進行底層系統重灌 (Restore)。**

**準備工作：電腦端工具選擇**
若您沒有 Mac 或未安裝 Apple Configurator，請依作業系統選擇替代工具：
*   **macOS (Catalina 後)**：直接開啟 **Finder**。
*   **Windows 10/11**：推薦下載 **「Apple 裝置 (Apple Devices)」App** (微軟商店最新版)，
    或使用傳統 **iTunes**。

**第一階段：復原模式 (Recovery Mode) - 適用於一般卡死**
*   **畫面特徵**：黑底螢幕，顯示一條傳輸線指向電腦圖示。
*   **操作方法 (iPad 9 / 具備 Home 鍵機型)**：
    1.  將 iPad 連接至電腦。
    2.  同時按住 **頂端電源鍵** + **Home 鍵** 不放。
    3.  裝置會強制重啟，看到 Apple 標誌也**不要放手**，直到出現「連接電腦圖示」才放開。
*   **電腦端操作**：軟體會跳出視窗，請選擇 **「回復 (Restore)」** (注意：這會清除所有資料)。

**第二階段：DFU 模式 (Device Firmware Update) - 適用於更嚴重的底層故障**
若復原模式失敗（例如刷寫到一半報錯），需進入更深層能徹底回復韌體的 DFU 模式。
*   **畫面特徵**：**完全黑屏**。若螢幕有任何圖示，代表失敗，請重來。
*   **操作方法 (iPad 9 / 具備 Home 鍵機型 - 強制切入法)**：
    1.  將 iPad 連接電腦（無論目前畫面為何）。
    2.  同時按住 **頂端電源鍵** + **Home 鍵** 不放。
    3.  待螢幕**完全變黑**（強制關機瞬間），心中默數 **5 秒**。
    4.  **放開電源鍵**，但繼續按住 **Home 鍵** 不放，心中默數 **10 秒**。
    5.  **電腦端判斷**：若 iPad 螢幕保持**全黑**，但電腦跳出「偵測到一台處於復原模式的 iPad」，即成功進入 DFU。
    6.  執行 **「回復 (Restore)」**。

**實務建議：專家提示**
*   **時機掌控**：進入 DFU 的關鍵在於「螢幕變黑後的 5 秒」。
    若超過時間才放開電源鍵，會變成普通重開機（看到蘋果標誌即失敗）。
*   **硬體判定**：若進入 DFU 後刷機進度條卡住不動，或反覆報錯 (Error 4013/4014/9)，
    很可能代表 **NAND Flash (硬碟)** 或 **CPU 脫焊**，請考慮報修。
