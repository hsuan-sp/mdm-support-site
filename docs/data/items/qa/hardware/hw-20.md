# Q: iPad 的 Touch ID 指紋辨識失靈，是按鍵壞了還是設定問題？

# Answer

**Touch ID 失效通常與「手指/感應器髒汙」有關，但若設定頁面出現「失敗」訊息，則為硬體故障。**

**自我檢測三步驟**：

1.  **清潔感應器**：
    *   Home 鍵表面的藍寶石玻璃容易沾染手汗與油汙。
        請使用酒精擦拭布清潔 Home 鍵表面，並擦乾手指。
2.  **刪除並重新註冊**：
    *   前往「設定」>「Touch ID 與密碼」。
    *   刪除舊指紋，嘗試新增一個新指紋。
    *   **關鍵徵兆**：若在新增過程中，系統直接跳出 **「無法完成 Touch ID 設定」** 或 **「失敗」**，
        這代表 Home 鍵內部的感應模組已損壞。
3.  **硬體報修**：
    *   若判定為硬體損壞，由於 Touch ID 模組與主機板是加密配對的，
        **無法透過坊間維修店單獨更換按鍵**（換了會失去指紋功能，僅剩回上頁功能）。
    *   **必須**送回 Apple 授權維修中心處理。對於精進方案裝置，這通常意味著需要更換整新機。
