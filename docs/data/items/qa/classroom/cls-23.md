# Q: 全班 iPad 發出音效擾亂課堂秩序，老師可以一鍵「全班靜音」嗎？

# Answer

**可以「一鍵靜音」，但無法「一鍵恢復音量」。**

**操作方式**：
在「課堂」App 的主畫面，點選 **「靜音 (Mute)」** 圖示。全班 iPad 的媒體音量會瞬間歸零。

**技術限制 (隱私考量)**：
*   **只能變小，不能變大**：Apple 為了防止惡用（例如管理員故意在大半夜調大音量嚇人），
    MDM 協定 **不允許** 遠端調大音量或解除靜音。
*   **恢復方式**：下指令靜音後，若要恢復聲音，學生必須 **自己手動** 按側邊音量鍵調高音量，老師無法代勞。
