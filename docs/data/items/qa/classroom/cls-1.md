# Q: 老師可以透過「課堂 (Classroom)」App 看到每位學生的螢幕嗎？

# Answer

**可以。這是「課堂」App 的核心功能，但具備嚴格的隱私提示機制。**

**功能運作與限制**：
1.  **即時監控**：老師在 iPad 或 Mac 上開啟「課堂」App，能看見全班學生的即時螢幕縮圖。
    點擊縮圖可放大檢視單一學生的操作細節。
2.  **隱私強制提示**：當老師正在查看螢幕時，學生 iPad 的狀態列會出現 **藍色的 AirPlay 圖示**（或藍色膠囊狀提示）。
    這是 iOS 系統層級的強制告示，無法透過 MDM 關閉，以保障學生知情權。

**技術先決條件**：
*   **藍牙與 Wi-Fi**：雙方裝置必須開啟藍牙與 Wi-Fi，且位於彼此的藍牙訊號範圍內（同個教室）。
*   **受監管裝置**：學生 iPad 必須是 **「受監管 (Supervised)」** 狀態，
    才能強制執行螢幕檢視而不需學生手動點擊「同意」。
*   **權限設定**：在 Jamf Pro 的限制描述檔中，必須勾選 **「允許 AirPlay 與螢幕檢視」**
    以及 **「強制 AirPlay 請求」**才能進行螢幕檢視和 Apple TV 投影。
