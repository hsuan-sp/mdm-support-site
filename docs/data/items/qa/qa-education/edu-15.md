# Q: 【定位爭議】開啟「遺失模式」會侵犯我的隱私嗎？

# Answer

**不會。遺失模式 (Lost Mode) 是專為尋回設備設計的，只有在設備被標記為「遺失」時才會啟用定位。**

**運作邏輯**：
1.  **強制開啟**：即使使用者平時關閉了定位服務，MDM 在下達「遺失模式」指令後會強制開啟定位功能。
2.  **使用者感知**：設備螢幕會鎖定並顯示「此 iPad 已遺失」以及管理員預設的聯絡電話。
3.  **隱私保護**：管理員只有在遺失模式下才能取得座標；一旦解除遺失模式，
    定位功能會恢復使用者原本的設定，管理員無法持續追蹤。
