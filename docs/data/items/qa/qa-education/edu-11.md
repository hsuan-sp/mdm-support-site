# Q: 「引導使用模式 (Guided Access)」跟「單一 App 模式」有什麼不一樣？

# Answer

**兩者都能將 iPad 鎖定在單一 App 中，但控制權不同：**
**引導使用模式由使用者（老師）手動開關；單一 App 模式由 MDM 強制設定，重開機也無法解除。**

這兩種鎖定方式經常被混淆，但用途與彈性完全不同。

**功能比較 (2026 版)**：

| 特性 | **引導使用模式** | **單一 App 模式 (ASAM)** |
| :--- | :--- | :--- |
| **啟動方式** | 手動（按三下電源/Home 鍵） | MDM 派發指令 或 App 自行觸發 |
| **主要用途** | 課堂防分心、幼教使用 | **高規格測驗 (CBT)**、數位看板 |
| **學生能否退出** | 可（若知道密碼） | **絕對無法自行退出** |
| **AI 工具控制** | 需手動調整 Accessibility | **自動禁用系統 AI** (如寫作工具) |
| **重開機後** | 解除（需重新啟動） | **仍被鎖定** |
| **需 MDM** | ❌ 不需要 | ✅ 需要 |
