# 術語定義
Screen Time（螢幕使用時間）是 Apple 提供的裝置使用監控與限制工具。它能記錄各類 App 的使用時長、設定 App 使用上限、或設定「停用時間」。MDM 可預先配置這些設定，協助老師管控課堂紀律，或防止學生在晚上過度使用平板。
# 白話文比喻
這是平板的「健康小管家」。它會幫你記帳，看你今天看影片看多久、玩遊戲玩幾分鐘。老師或家長可以跟管家說：『每天只能玩一小時，時間到了就把門關上』。這樣就能幫學生養成良好的數位使用習慣，不會玩到廢寢忘食。
