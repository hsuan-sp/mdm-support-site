# 術語定義
iOS 26/iPadOS 26 DDM功能，組織可限制 App 只能透過 Wi-Fi 下載，禁止使用行動網路，節省數據流量。

# 白話文比喻
就像家長設定「遊戲只能在家裡的 Wi-Fi 玩，不能用手機流量」，避免學生用光流量。

# MDM 相關
精進方案 iPad 若有插 SIM 卡（部分縣市有配發），可防止學生用行動網路下載大型 App 或遊戲。
