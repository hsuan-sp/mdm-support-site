# 術語定義
Passcode Policy（密碼原則）是企業針對行動裝置解鎖密碼所設定的強制規範。MDM 可設定要求的密碼複雜度（如：最少位數、是否包含字母、不可包含重複字元）以及密碼到期天數。若裝置不符合規範，系統會強制要求使用者在設定時更新密碼才能繼續使用。
# 白話文比喻
這是家裡的「大門密碼規定」。為了安全，公司規定密碼不能設「0000」或「1234」，必須得有 6 位數以上，還要定期更換。這就像是規定每三個月就要換一次門鎖密碼一樣，雖然有點麻煩，但能確保萬一裝置弄丟了，裡面的機密資料也不會被外人輕易解鎖。
