## 術語定義

在教育科技領域，**Roster (班級名冊)** 是定義學生、教師與班級之間關連性的數位資料集。

資料流向：

1.**來源**：源自學校的 **學生資訊系統 (SIS)**(如 PowerSchool 或校務系統)。

2.**同步**：透過 SFTP 或 API 上傳至 **Apple School Manager (ASM)**。

3.**應用**：自動匯入 **Apple Classroom (課堂)** 與 **Schoolwork (功課)** 等 App。

準確的名冊能確保當老師走進教室打開 iPad 時，完全不用手動輸入，就能自動看到「七年級數學課」座位上那 25 位正確的學生名單。

## 白話文比喻

這是 **「數位點名簿」**。

* **手動**：以前老師要自己建 Excel 表格，一個字一個字把學生名字打進去。
* **自動推送**：現在，學校的中央電腦系統會自動把最新的名單 **「推」** 到老師的 iPad 裡。
* **結果**：當王老師走進生物課教室，他的 iPad 早就知道這節課該有哪些學生，一個都不會少，也不會打錯名字。
