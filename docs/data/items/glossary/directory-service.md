## 術語定義

**Directory Service (目錄服務)** 是企業集中儲存與管理使用者身分、電腦資訊與存取權限的資料庫系統。

常見的服務包括 Microsoft Active Directory (AD) 或 LDAP。透過與 MDM 的整合，企業可以達成：

* **單一登入**：員工可直接使用既有的公司帳號登入 iPad 或 Mac。
* **權限同步**：依據員工所在的部門自動派發對應的 App 與設定。
* **統一管理**：員工離職時，只需停用目錄帳號，所有權限自動失效。

## 白話文比喻

這是全公司的 **「戶籍資料庫」**。

裡面詳細記錄了每個員工是誰、住在什麼部門、官階權限是多少。

其他系統（如 MDM、門禁系統、Email）要查身分時，都會來問這個 **中央資料庫**，確保大家的資料都是一致的，不會發生「人事部已經把人開除了，但他還能刷卡進公司」的漏洞。
