## 術語定義

**Extension Attribute (擴充屬性)** 是 Jamf Pro 提供的強大客製化功能。

標準的 MDM 只能收集預設資訊（如序號、型號、OS 版本）。EA 允許管理員透過 **Script (Bash/Python)** 或 **輸入欄位**，收集任何你想知道的額外資訊，例如：

* **資安檢查**：偵測電腦是否安裝了非法的挖礦軟體或遊戲。
* **硬體狀態**：檢查電池的循環次數或健康度。
* **檔案驗證**：確認某個特定的內部設定檔是否存在於系統中。

這些收集回來的資料，都可用來建立 **智慧型群組 (Smart Group)** 進行進階管理。

## 白話文比喻

這是資產盤點時的 **「自訂問卷欄位」**。

一般的盤點表只會問：「你叫什麼名字？」、「身高體重多少？」（標準屬性）。

Extension Attribute 讓你可以在問卷下面的空白處 **自己加題目**，例如：「你有沒有偷打電動？」、「你昨天幾點睡覺？」。這樣老師（管理員）就能掌握更細節、更個人化的學生狀況。
