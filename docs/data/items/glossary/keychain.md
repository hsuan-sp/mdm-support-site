## 術語定義

**Keychain (鑰匙圈)** 是 Apple 裝置內建的整合式密碼管理系統。

它負責安全地儲存與管理：

* **網站與 App 密碼**：自動記住並填寫登入資訊。
* **數位憑證**：用於連接學校 Wi-Fi (802.1X) 或 VPN 的身分證明。
* **私密金鑰**：用於加密通訊的核心資料。

一旦使用者解鎖裝置，Keychain 就會自動解鎖，讓使用者無需反覆輸入密碼即可通行無阻。

## 白話文比喻

這是系統的 **「萬能鑰匙包」** 或 **「隨身保險箱」**。

你不用把幾十把鑰匙（各種密碼）掛在脖子上叮叮噹噹響，而是全部鎖在這個加密的盒子裡。

只要你用指紋或開機密碼打開這個盒子，系統就會自動幫你拿出對的那把鑰匙去開門（登入網站或連 Wi-Fi），你甚至感覺不到它的存在。
