## 術語定義

**iCloud Backup (iCloud 備份)** 是 Apple 的自動化雲端備份服務，能將裝置的資料與設定複製到安全的雲端伺服器。

觸發自動備份的條件包括：

* **連線**：裝置必須連接 Wi-Fi。
* **電源**：裝置必須正在充電。
* **狀態**：螢幕必須處於鎖定狀態（通常發生在夜間充電時）。

**MDM 策略**：學校可透過 MDM 強制要求學生開啟備份以保護作業資料，或是因應隱私法規（如資料不得出境）而強制停用此功能。

## 白話文比喻

這是裝置的 **「自動雲端保險櫃」**。

每天晚上，當學生的 iPad 在睡覺（鎖定）且充電時，它會悄悄地把所有的作業、照片和設定打包一份，存到雲端的保險櫃裡。

萬一第二天 iPad 摔壞了，IT 只要給學生一台新機器，登入後，所有的 **「數位人生」** 都會原封不動地回來，就像什麼意外都沒發生過一樣。
