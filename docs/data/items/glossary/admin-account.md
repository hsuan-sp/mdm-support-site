## 術語定義

**Admin Account (管理者帳號)** 是 macOS 中的一種權限等級，賦予使用者對系統的完整控制權。

擁有管理者權限的使用者可以執行以下操作：

* **安裝軟體**：在系統層級新增或移除應用程式。
* **修改設定**：變更網路配置、安全性政策以及全系統的偏好設定。
* **管理使用者**：建立、刪除其他本機帳號，或重設其密碼。

在受管理的環境中，資訊部門通常會保留一個 **隱藏的管理員帳號** 供維護使用，並將一般使用者的權限設定為 **「標準使用者 (Standard User)」**，以防止裝置被進行未經授權的變更。

## 白話文比喻

這就是電腦的 **「最高指揮官」**。

持有這個帳號的人可以隨意更改任何設定、刪除檔案或安裝軟體。為了系統安全，通常只有資訊人員具備此等級的權限。

其他人則被賦予 **「標準帳號」**，這代表他們可以順暢地使用電腦完成日常工作，但無法隨意更動「引擎蓋下」的核心系統引擎。
