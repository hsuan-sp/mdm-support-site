## 術語定義

**802.1X** 是 IEEE 制定的一種連接埠存取控制 (Port-based Network Access Control, PNAC) 標準。它為希望連上區域網路 (LAN) 或無線網路 (WLAN) 的裝置，提供了強大的身分驗證機制。

在企業與校園環境（如 **eduroam**）中，802.1X 要求：

* 裝置必須部署有效的**數位憑證** (Digital Certificates) 或帳號密碼。

* 在獲准存取網路前，這些憑證必須先經過 **RADIUS 伺服器**的驗證。

這比共用 Wi-Fi 密碼的方式安全得多，因為它確保了只有擁有唯一身分證明且經過授權的裝置，才能進入內部網路。

## 白話文比喻

想像這就像是您學校網路的**「機場安檢站」**。

* **一般家用 Wi-Fi** 就像是一把家裡的鑰匙——任何人只要拿到了鑰匙，就能直接開門進去。

* **802.1X** 則要求每個人都必須出示自己的**護照**（憑證）與**機票**（認證資訊）。

如果身分與安全名單不符，他們就會被擋在閘門外，無法進入。
