# 術語定義
iPadOS 26.2 強化的拖放式多工手勢。使用者可直接從 Dock 或 Spotlight 拖拉 App 圖示，快速建立分割視圖（Split View）、滑動置前（Slide Over）或視窗配置，無需繁瑣的多步驟操作。

# 白話文比喻
就像整理書桌時，直接把書本從書架（Dock）拖到桌上排好位置，不用再一本一本「複製貼上」才能並排閱讀。
