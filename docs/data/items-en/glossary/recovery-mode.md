## Definition

Recovery Mode is a special built-in state on Apple devices used for troubleshooting and system maintenance. It allows users to reinstall the operating system, repair disk errors, or restore the device from a backup when the standard system fails to start correctly. On modern Macs, entering Recovery Mode requires a specific physical gesture (long-pressing the power button) and may be protected by a **Recovery Lock**.
