## Definition

Lightweight Directory Access Protocol (LDAP) is an open industry-standard protocol for accessing and maintaining distributed directory information services over an IP network. MDM systems use LDAP to "talk" to institutional databases (like Active Directory) to retrieve lists of students, staff, and departments. This allows IT to target specific management policies to the correct group of people automatically.
