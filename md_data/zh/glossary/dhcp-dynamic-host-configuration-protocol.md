## 術語定義

**DHCP (Dynamic Host Configuration Protocol)** 是一種自動分配 IP 位址的網路協定。

其運作流程如下：

1. **發送請求** ：當裝置連上網路時，會大喊「我需要一個 IP！」。

2. **自動分配** ：DHCP 伺服器聽到後，會自動 **租借** 一組閒置的 IP 給它使用。

3. **回收再用** ：當租約到期或裝置斷線後，這組 IP 會被回收，清洗乾淨後再分配給下一台裝置。

這避免了 IT 人員手動設定 IP 的麻煩與衝突風險。

## 白話文比喻

這是網路停車場的 **「車位分配員」** 。

當你開車進入停車場（連上網路），管理員會自動指派一個 **號碼牌 (IP)** 給你停。

你不用自己找車位，也不會發生兩台車搶同一個車位的尷尬狀況。當你開車離開後，管理員就把那個號碼牌收回來，留給下一台進場的車使用。
