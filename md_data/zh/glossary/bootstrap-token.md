## 術語定義

**Bootstrap Token (引導代幣)** 是 macOS 用於安全啟動與加密管理的自動化授權憑證。

運作機制：

1. **產生** ：當第一位具備安全權限的使用者啟用 **FileVault** 加密時，系統會自動產生此代幣。

2. **託管** ：代幣會安全地託管給 **MDM 伺服器** 。

3. **授權** ：MDM 可使用此 Token 授權其他管理員帳號登入已加密的 Mac，或在 **無人值守** 的情況下自動完成軟體更新的授權驗證 (Secure Token)。

## 白話文比喻

這是由 MDM 替你保管的 **「系統備用萬能鑰匙」** 。

當電腦被 **FileVault** 加密鎖住時，通常只有原使用者的鑰匙打得開。如果不小心把鑰匙弄丟了，或者需要派另一個 IT 人員進去維修，這時候 MDM 手上的 **Bootstrap Token** 就能發揮作用。

它讓 IT 人員不需要求助於原本的使用者，就能隨時 **「開門」** 進行維護，或是把鑰匙複製一份給新來的員工使用。
