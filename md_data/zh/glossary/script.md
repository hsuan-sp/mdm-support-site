## 術語定義

**Script (腳本)** 是一段在受管 Mac 上執行的客製化程式碼（通常是用 Bash, zsh 或 Python 寫的）。

它的威力：

- **超越 MDM** ：MDM 描述檔通常只有固定的「勾選框」（例如「開啟 Wi-Fi」），但 Script 讓 IT 可以透過程式碼叫電腦做 **任何** 它能做的事。

- **自動化** ：可以自動修復壞掉的印表機設定、刪除暫存檔，或修改隱藏的系統參數。
- **擴充收集** ：用來計算並回傳 MDM 原本收集不到的資訊（擴充屬性）。

## 白話文比喻

請把它想像成 **「特殊的魔法咒語」** 。

- **一般按鈕** ：控制台裡只有「音量」、「亮度」這種普通按鈕。但有時候你會遇到一些怪問題，是沒有按鈕可以解決的。

- **咒語** ：這時候，IT 巫師就會寫一段專屬的 **「咒語」** (Script) 並命令電腦唸出來。電腦照著咒語執行複雜的動作，就能完成那些普通按鈕做不到的神奇修復任務。
